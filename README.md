# Sound API lab

## Admission Number
150767

## Upload file type
Only .mp3 files can be uploaded

## Interactivity
Green when playing
Red when paused, seeking or end of a song

## Keyboard controls
Keyboard controls:
    arrowUp: volume up
    arrowDown: volume down
    arrowLeft: go back 5 seconds
    arrowRight: go forward 5 seconds
